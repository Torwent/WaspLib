Data

Methods to fetch osrs data. This file has methods to fetch information about items, monsters and weapons.

const PRICES_URL

Global PRICES_URL constant. Can be changed to use a different prices data json as long as it has follows the wiki prices structure.

const OSRSBOX_URL

Global OSRSBOX_URL constant. Can be changed to use a different items data json as long as it has follows OSRSBox’s structure.

type TRSItemData

TRSItemData is the record used to handle fetching item data.

ItemData.Free()

procedure TRSItemData.Free();

Internal method called automatically on script termination. Shouldn’t be called by users other wise another termination will be attempted when the script stops running.

ItemData.SetupWikiPrices()

procedure TRSItemData.SetupWikiPrices();

Internal method called automatically when we require price data. Doesn’t need to be called since it will be called automatically if TRSItemData.WikiPriceIsSetup is false. It’s also recalled when we try to fetch item data and 20 minutes have passed to update prices.

ItemData.GetIDs

function TRSItemData.GetIDs(Item: String): TStringArray;

Internal method called automatically when we require item data. This is based of ItemFinder.GetItemIDs() and it’s used to retrieve item ids.

ItemData.GetData()

function TRSItemData.GetData(Item: String): TJSONObject;

Internal method used to retrieve a TJSONObject of the Item we want information on. Item should be an item ID.

ItemData.GetTradeableID()

function TRSItemData.GetTradeableID(Item: TRSItem): String;

Internal method used to retrieve only the tradeable version of an item if it exists. Some items have versions of themselves that are not tradeable and can’t be used to fetch item prices, such as degraded items, ornamented items, etc.

ItemData.GetPrice()

function TRSItemData.GetPrice(Item: TRSItem): TJSONObject;

Internal method used to fetch a price data TJSONObject of a certain item.

ItemData.GetPriceInt()

function TRSItemData.GetPriceInt(Item: TRSItem; Key: String): Int32;

Wrapper method used to fetch ints from TRSItemData.GetPrice(). To use this you should be familar with the structure of the JSONObject TRSItemData.GetPrice() returns.

Example

WriteLn ItemData.GetPriceInt('Abyssal whip', 'low'); // will print the low price of abyssal whip.
WriteLn ItemData.GetPriceInt('Abyssal whip', 'high'); // will print the high price of abyssal whip.
