ScriptForm.Init¶

procedure TScriptForm.Init(Sender: TObject);

Method used to load the variables set in the Script form to be used by our script. Unless you are using a vanilla form it’s likely you will want to override this.

ScriptForm.Run¶

procedure TScriptForm.Run();

Method used to run the TScriptForm. This is meant to be overriden and call TScriptForm.Setup(). The reason it doesn’t call it already is because users are likely to modify TScriptForm to their needs and TScriptForm.Setup() needs to be called in the user script to load his changes.
