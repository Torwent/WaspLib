Magic

Extends SRL’s TRSMagic.

Magic._CastedSpell

function TRSMagic._CastedSpell(spell: ERSSpell): Boolean;

Internal helper function for better logic of wether we casted a spell or not. Used by WaspLib’s TRSMagic.CastSpell() override.

And yes, I know casted is not a word.

Magic.CastSpell

function TRSMagic.CastSpell(spell: ERSSpell): Boolean; override;

Overrides the existing SRL TRSMagic.CastSpell() for better cast detection with the usage of TRSMagic._CastedSpell().
