ItemFinder¶

Methods to extend SRL’s ItemFinder functionality.

This file is responsible for loading fixed sprites for known broken items with the default SRL (e.g.: Rune pouch).

Some handy item wrappers were added for all cakes, pies and pizzas (e.g.: Cake(3) for “Cake”, Cake(2) for “2/3 cake” and Cake(1) for “Slice of cake”).

const WL_ITEM_FINDER_IMAGES_PATH¶

Global const for WaspLib’s ItemFinder override images: Simba\Includes\WaspLib\osr\itemfinder\images\

ItemFinder.GetItemImages¶

function TRSItemFinder.GetItemImages(Item: String): array of TMufasaBitmap; override;

Overrides the default ItemFinder.GetItemImages so it also loads the fixed sprites stored in WL_ITEM_FINDER_IMAGES_PATH.
