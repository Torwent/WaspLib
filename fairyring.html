FairyRing.Teleport

function TRSFairyRing.Teleport(code: String): Boolean;

Teleports the user to the specified fairy ring code. If the fairy ring interface is not open, the RSObject will be used to find the closest one and open it. If the code is on the right click context menu, it will be used.

Example

WriteLn FairyRing.Teleport('cip');

FairyRing.WalkTeleport

function TRSFairyRing.Teleport(code: String): Boolean;

Teleports the user to the specified fairy ring code. If the fairy ring interface is not open, the RSObject will be used to find the closest one and open it. If the fairy ring is not visible, we will walk to it. If the code is on the right click context menu, it will be used.

Example

WriteLn FairyRing.Teleport('cip');
